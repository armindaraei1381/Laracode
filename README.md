## 🇺🇸 English

### Description
Laracode is a minimalist and powerful IDE designed specifically for PHP and Laravel developers. It streamlines your workflow by combining essential tools into a single, native application.

### Features
*   **Project Management**: Create, open, and manage your Laravel projects effortlessly.
*   **Built-in Terminal**: Integrated native terminal for executing Artisan commands and scripts.
*   **File Explorer**: Intuitive sidebar for easy file navigation and editing.
*   **Server Management**: Start and stop your local development server with one click.
*   **Minimalist UI**: Focused on productivity with a clean One Dark Pro-inspired design.

<a name="فارسی"></a>
## 🇮🇷 فارسی

### معرفی
لاراکد (Laracode) یک محیط توسعه (IDE) مینیمال و قدرتمند است که به‌طور اختصاصی برای برنامه‌نویسان PHP و فریم‌ورک Laravel طراحی شده است. این ابزار با تجمیع تمام نیازهای شما در یک محیط واحد، سرعت توسعه را افزایش می‌دهد.

### ویژگی‌ها
*   **مدیریت پروژه**: ایجاد، باز کردن و مدیریت آسان پروژه‌های لاراول.
*   **ترمینال داخلی**: ترمینال بومی یکپارچه برای اجرای دستورات Artisan و اسکریپت‌ها.
*   **فایل اکسپلورر**: دسترسی سریع و مدیریت آسان فایل‌های پروژه.
*   **مدیریت سرور**: راه‌اندازی و توقف سرور لوکال تنها با یک کلیک.
*   **رابط کاربری مینیمال**: طراحی ساده و چشم‌نواز با الهام از تم محبوب One Dark Pro جهت تمرکز بیشتر بر کدنویسی.
