# Contributing to Laracode

First off, thanks for taking the time to contribute! 🎉

### How to contribute:

**1. Report Bugs:**
If you find a bug, please open an **Issue** and include:
- A clear description of the problem.
- Steps to reproduce it.
- Screenshots if possible (you can use our assets folder style).

**2. Suggest Features:**
Got an idea to make Laracode better? Open an **Issue** with the tag "Feature Request" and let's discuss it!

**3. Submit Code (Pull Requests):**
- Fork the repository.
- Create a new branch for your feature/fix.
- Follow the existing **One Dark Pro style** and clean Python coding practices.
- Submit a Pull Request. I will review it as soon as possible.

### Code Style
- Keep it clean, minimalist, and readable.
- Use English comments for your code.
- Ensure your changes don't break the existing Laravel-focused functionality.

---
*Happy Coding!*
