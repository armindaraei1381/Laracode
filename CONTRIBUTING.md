# 🚀 Contributing to Laracode

First off, thank you so much for taking the time to contribute to Laracode! 🙌✨ Your help makes this project better for everyone. 🛠️

---

### 💡 How can you help?

**1. 🐛 Report Bugs:**
Found something acting weird? Please open an **Issue** and include:
- 📝 A clear description of the problem.
- 👣 Steps to reproduce it.
- 📸 Screenshots (if possible, to help me visualize!).

**2. 💡 Suggest New Features:**
Have a cool idea to make Laracode even better for Laravel/PHP devs? 🌟 Open an **Issue** with the label `Feature Request` and let's brainstorm together! 🧠💭

**3. 💻 Submit Code (Pull Requests):**
Want to dive into the code? Awesome! ⚡
- 🍴 **Fork** the repository.
- 🌿 Create a new **branch** for your feature or fix.
- 🎨 Keep the **One Dark Pro** spirit and clean Python practices alive.
- 📤 Submit a **Pull Request**. I'll review it with excitement! 🧐🚀

---

### 📏 Code Style Guidelines
- 🧼 Keep it **clean, minimalist, and readable**.
- 🇬🇧 Use **English comments** throughout your code.
- 🎯 Ensure your changes respect the **Laravel-focused philosophy** of this IDE.

---

### 💖 Join the community!
Whether you're fixing a typo or adding a complex feature, every contribution is gold. 🥇✨ Let's build the best lightweight IDE for PHP together! 🐘💻🔥

*Happy Coding!* 👩‍💻👨‍💻✨
